# Manutencao-Preditiva-em-Maquinas-Industriais
Análise de dados operacionais coletados por 7 meses de 3 máquinas de alta precisão (setores aeroespacial, automotivo e médico) com foco em predição de falhas. Projeto envolve tratamento e exploração com Pandas, modelagem com Scikit-learn e identificação de padrões para apoiar manutenção preventiva e reduzir downtime.
